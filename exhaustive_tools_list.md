# Comprehensive Directory of Browser-Memory Client-Side Tools (50+ Utilities)

This is an exhaustive directory of popular, daily-use web tools that users search Google for, but which can be run **100% locally and offline** in the browser sandbox.

---

## 1. Document & PDF Tools
* **PDF Merge & Split:** Combine multiple files or extract page ranges.
* **PDF Page Organizer:** Drag-and-drop to reorder, delete, or rotate pages.
* **PDF Password Remover/Adder:** Decrypt or encrypt documents locally.
* **Interactive Invoice & Receipt Builder:** Live forms calculating totals and printing to PDF.
* **EPUB to PDF Converter:** Convert EPUB books to PDF.
* **PDF Text Extractor:** Strip selectable text lines from standard PDFs.
* **PDF Image Extractor:** Extract raw embedded JPG/PNG assets from a PDF document.
* **Word to PDF / HTML to PDF:** Compile rich layouts or documents into PDFs.

## 2. Image & Vector Graphic Utilities
* **ICO Converter (Favicon Maker):** Package PNG sizes into a single `.ico` file.
* **Eyedropper Color Picker:** Pixel magnifier to read HEX/RGB codes from images.
* **Meme & GIF Annotator:** Overlay styled text lines on canvas or animated GIFs.
* **Sprite Sheet Packer:** Bin-pack multiple images and export combined image + coordinate JSON.
* **ASCII Art Converter:** Translate pixels into custom text art maps.
* **SVG Path Editor:** Adjust path handles and vector points on a grid.
* **SVG Optimizer / Minifier:** Clean redundant XML tags and compress SVG files.
* **Image-to-Base64 / Base64-to-Image:** Encode pictures to inline tags or decode back to files.
* **EXIF Metadata Strip:** Wipe camera metadata and GPS tags.

## 3. Video, Audio & Screen Captures
* **Video Trimmer / Cutter:** Set crop markers to trim local MP4/WebM videos.
* **Video to GIF Converter:** Compile video frames to animated GIF loops.
* **Video Audio Extractor:** Rip the audio track from videos as MP3/WAV.
* **Audio Cutter & Waveform Editor:** Trim and fade MP3 audio using WaveAudio APIs.
* **Audio Format Converter:** Transcode audio files (WAV, MP3, OGG).
* **Audio Waveform SVG Renderer:** Convert audio files into podcast wave graphics.
* **Offline Screen & Voice Recorder:** Capture screen share viewport or microphone recordings using WebRTC.

## 4. Text Manipulation & Writing Tools
* **Word & Character Counter:** Real-time counting of letters, sentences, paragraphs, and reading times.
* **Case Converter:** Change text blocks to UPPERCASE, lowercase, Title Case, or Sentence Case.
* **Case & Duplicate Line Remover:** Strip matching duplicate lines from directories or text files.
* **List Randomizer & Sorter:** Sort arrays alphabetically, numerically, or shuffle items.
* **Find and Replace Utility:** Search and substitute text blocks with regex support.
* **HTML Tag Stripper:** Remove rich tags and export clean, raw plain-text.
* **Lorem Ipsum Generator:** Generate dummy placeholder text paragraphs.
* **Morse Code Converter:** Translate text blocks to and from dot-dash Morse code.

## 5. Security & Cryptographic Generators
* **Random Password Generator:** Custom length passwords with checkboxes for numbers, symbols, and cases.
* **Passphrase Generator:** Combine random dictionary words into secure, memorable passphrases.
* **Password Strength Meter:** Local audits of password complexity and entropy.
* **Symmetric Encrypter/Decrypter:** Encrypt text using passwords via AES, DES, or TripleDES.
* **Hash / Checksum Calculator:** Compute SHA-256, SHA-512, or MD5 sums from local files.
* **JWT Decoder:** Parse JWT header and payload sections.

## 6. Developer Formatters & Translators
* **Diff Checker:** Visual, side-by-side text comparisons with color-coded diff highlights.
* **JSON Prettifier & Validator:** Format raw strings with syntax highlighting and syntax error alerts.
* **JSON to YAML / YAML to JSON:** Convert configuration files.
* **CSV to JSON / JSON to CSV:** Translate tabular datasets.
* **RegEx Match Tester:** Regex group match testing workbench.
* **Epoch / Unix Timestamp Converter:** Translate integers into date-time strings and vice versa.
* **Cron Job Expression Builder:** Graphically construct cron statements and preview runs.
* **Javascript/HTML/CSS Minifier:** Strip spaces and comments to compress source code.

## 7. Date, Time & Mathematics
* **Age Calculator:** Compute age in years, months, weeks, days, hours, and seconds.
* **Days Calculator:** Calculate exact day differences, include/exclude weekends, or add/subtract days.
* **World Clock & Timezone Converter:** scrubbing slider to map meeting alignments across added global zones.
* **Scientific Calculator:** Offline scientific computations.
* **Radix Converter (Hex/Dec/Bin):** Translate numeric values between binary, octal, decimal, and hexadecimal.
* **BMI Calculator:** Calculate body mass index ratios.
* **Loan / EMI Amortization Calculator:** Compute interest schedules and graph repayments.
* **Percentage Calculator:** Fast percentage increments, fractions, and calculations.

## 8. Codes & Scanners
* **QR Code Generator:** Convert URLs, text, or Wi-Fi configurations into QR codes.
* **QR Code Reader:** Scan QR codes from uploaded images or live webcams.
* **Barcode Generator:** Output product barcodes (UPC, EAN, Code128).
* **Barcode Reader:** Decode product numbers from images.

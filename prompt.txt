# Build ImageCraft AI

You are a Staff-Level Software Architect, Senior Frontend Engineer, UX Designer, SEO Specialist, and Performance Engineer.

Build a complete production-ready web application called "ImageCraft AI".

The application must be a collection of privacy-first image tools that run entirely inside the user's browser.

## Primary Goal

Create an MVP that:

* Requires no backend
* Requires no database
* Requires no cloud processing
* Requires no paid APIs
* Requires no OpenAI API
* Requires no Gemini API
* Requires no AWS services
* Requires no Azure services

The application must be deployable as a static website.

All image processing and AI inference must run entirely in the user's browser.

## Business Goal

Generate organic traffic through SEO and monetize using Google AdSense.

## Core Value Proposition

"Your files never leave your device."

Display this message prominently throughout the application.
